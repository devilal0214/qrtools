QR Code Logo Download Test

Test Instructions

  1. Go to http://localhost:3001
  2. Create a QR code with any content
  3. In the "Design Your QR" section, upload a PNG or SVG logo
  4. Click "Download QR" and test both PNG and SVG formats
  5. Verify that the downloaded files contain both QR code and logo

Expected Behavior

  • PNG Download: Should include QR code with logo overlay
  • SVG Download: Should include QR code with logo as embedded image or shape
  • No Logo: Should download clean QR code without logo elements

Debug Information

Open browser developer console to see debug logs during download:

  • SVG export debug: - Shows logo detection status
  • PNG export debug: - Shows PNG conversion process
  • Logo successfully included: - Confirms successful injection

Common Issues to Check

  • Logo not showing: Check if logoImage or logoPreset values are set
  • PNG only shows logo: html2canvas fallback should create complete image
  • SVG missing logo: Logo markup injection failed
  • Console errors: Image loading or canvas rendering issues
